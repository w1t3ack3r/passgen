# Strong Password Generator

A web-based application that generates strong passwords based on user input. This tool aims to help users create secure passwords, enhancing their online security.

## Features
- **Customizable Input**: Users can enter a word or phrase (8+ characters) to generate a password.
- **Strong Password Generation**: The app generates passwords with character replacements and adds random numbers for increased security.
- **User-Friendly Interface**: Built with Bootstrap for a clean and responsive design.
- **Error Handling**: Provides validation for user input to ensure the creation of strong passwords.
- **Copy to Clipboard**: Easily copy generated passwords with a single click.

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap 5
- Font: Montserrat (Google Fonts)

## Usage
- Enter a word or phrase with at least 8 characters.
- Click on the "Generate Password" button.
- Your generated password will be displayed.
- You can copy the password to your clipboard or regenerate a new password as needed.
